# Kaggle API Setup

1. Go to [Kaggle.com](https://kaggle.com) and sign in
2. Click on your profile → "Settings"
3. Scroll to "API" section → "Create New API Token"
4. Save the downloaded `kaggle.json` file
5. Place it in `~/.kaggle/` (Mac/Linux) or `C:\Users\<username>\.kaggle\` (Windows)
6. Run: `chmod 600 ~/.kaggle/kaggle.json` (Mac/Linux only)

Then run the notebook - it will automatically download the dataset.
