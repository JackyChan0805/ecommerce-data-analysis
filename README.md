# ecommerce-data-analysis
E-commerce Data Analysis Project - Customer Behavior &amp; Sales Insights

## Project Overview
This project analyzes an e-commerce dataset to extract actionable business insights about sales patterns, customer behavior, and product performance.

## Key Analyses
- Sales trends by month and weekday
- Top-performing products identification
- Customer segmentation (high/middle/low spenders)
- Geographic sales distribution
- Business recommendations

## Key Findings
- Peak Sales: November and Thursday show highest sales
- Top Product: "PAPER CRAFT , LITTLE BIRDIE" (£168K)
- Core Market: UK contributes 82% of total sales
- Customer Value: Top 10% customers drive 61% of revenue

## Technologies Used
- Python 3.12
- Pandas, NumPy
- Matplotlib, Seaborn
- Jupyter Notebook
- Kaggle API

## How to Run
1. Clone this repository
2. Install dependencies: `pip install -r requirements.txt`
3. Set up Kaggle API (see instructions)
4. Run the data_clean.py to get the cleaned_ecommerce_data.csv
5. Run the Jupyter notebook

## Dataset
Source: [Kaggle - E-commerce Data](https://www.kaggle.com/datasets/carrie1/ecommerce-data)
